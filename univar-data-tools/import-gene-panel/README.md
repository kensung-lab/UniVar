# Import Gene Panel

## Description
This is a [Typescript][type-script] program to import all [gene panels][gene-panel] within the `data` directory. 
For downloading [gene panels][gene-database] to the `data` directory, please refer to the [data_script](data_script) folder.
Also, please create a `.env` file in the root directory and having the `MONGO_BASE_URL` with the mongoDB that you want to connect.

The `panel2hpo_top5.tsv` is generated by [panel2hpo](./panel2hpo/). It contain the top 5 [HPO terms][hpo-website] that related to that [gene panel][gene-panel].

After gather panel2hpo_top5.tsv please copy to the `data` directory.

## Reminder
Please check the version number on top of the program before every time running it
```javascript
// versioning (Update EVERY TIME !!!!!)
const UNIVAR_GENE_PANEL_VERSION = '1.0.1';
const CLINGEN_PANEL_VERSION = '05-05-2023';
const PANEL_UK_VERSION = '05-05-2023';
const PANEL_AU_VERSION = '05-05-2023';
const CUSTOM_PANEL_VERSION = '1.0.0';
```

## Requirement
1. [NodeJS][node-js] 18+

## Installation
```bash
npm install -g pnpm # if and only if not installed pnpm
pnpm install
```

## Compile
```bash
pnpm run build
```

## Run
```bash
pnpm run start
``` 

## Example of .env file
```.env
MONGO_BASE_URL=mongodb://18.163.142.246:27018/
```
